## 專案簡介
一個可線上查看活動並報名的網頁應用。具即時表單驗證、動態活動卡片與報名統計功能。可使用 localStorage 儲存報名記錄，並支援匯出 JSON。

## 使用技術
- HTML5（語意化標籤）
- CSS3 + Bootstrap 5（RWD）
- JavaScript（DOM 操作、事件監聽）
- Constraint Validation API（表單驗證）
- localStorage（加分）

## 功能特色
1. 活動列表以 JavaScript 動態產生（createElement）。
2. 表單使用 HTML5 驗證並提供自訂錯誤提示（was-validated）。
3. 即時報名預覽與總人數統計。
4. 防止重複送出（按鈕禁用）與匯出報名紀錄功能。
5. 支援深色模式（Ctrl + D 切換）。
