# 🍕 Pizza Builder! - 自訂披薩點餐模擬 (114_web_midterm_project)

## ✨ 專案簡介 (Project Introduction)

本專案是一個互動式的「自訂披薩點餐模擬器」，旨在整合 HTML、CSS、Bootstrap 與 JavaScript 等前端核心技術。使用者可以透過直覺的介面選擇披薩的尺寸與配料，系統會即時計算價格，並在視覺預覽區動態呈現選擇結果，最終透過表單驗證完成模擬下單流程。

## 🛠️ 使用技術 (Technology Stack)

* **結構：** HTML5 (使用語意化標籤)
* **樣式與排版：** CSS3, **Bootstrap 5** (實現響應式設計 RWD)
* **互動核心：** **原生 JavaScript (Vanilla JS)**
* **DOM 操作：** 使用 `querySelector`、`createElement`、`appendChild` 實現配料動態增減。
* **表單驗證：** HTML5 Constraint Validation API, 自訂錯誤提示 (配料數量檢查)
* **版本控制：** GitHub

---

## 🎯 專案功能特色 (Key Features)

1.  **即時價格計算 (Real-time Calculation)：** 根據選擇的尺寸和配料，系統會立即更新畫面上顯示的總金額。
2.  **動態視覺預覽 (Dynamic Visual Preview)：** 使用者點選配料時，會利用 DOM 操作在披薩基底上動態新增對應的配料圖示。
3.  **嚴格表單驗證 (Form Validation)：**
    * 結帳表單包含必填欄位和 Email 格式檢查。
    * **自訂驗證：** 必須選擇至少三種披薩配料才能送出訂單。
4.  **響應式設計 (RWD)：** 使用 Bootstrap 網格系統，確保在桌機、平板和手機上都能流暢地使用。
